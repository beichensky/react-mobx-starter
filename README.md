# react-mobx-starter
进行 React + Mobx + React Route 开发的脚手架


## 下载项目
``` bash
git clone https://github.com/beichensky/react-mobx-starter.git
```

## 安装插件
``` bash
npm install
```


## 运行项目
``` bash
npm run start
```

打开浏览器，输入：`localhost:8000` 查看项目运行效果



## 项目打包
``` bash
npm run build
```

新增 `dist` 目录，里面是打包好的项目代码
